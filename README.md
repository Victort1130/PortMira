# PortMira

**PortMira — A local-first portfolio mirror.**

PortMira is a local-first personal portfolio dashboard that lets you track assets, liabilities, and net worth — without sending your data anywhere.

---

## Current Stage

`v0.1.0-dev` — Project structure initialized. Under active development.

---

## Tech Stack

| Layer | Tool |
|---|---|
| Language | Python 3.11+ |
| UI | Streamlit |
| Data Processing | Pandas |
| Stock / ETF Prices | yfinance |
| Crypto Prices | CoinGecko API |
| FX Rates | Free FX API / fallback table |
| Charts | Plotly / Matplotlib / Streamlit |
| Storage | Local JSON / CSV |

---

## Privacy

All portfolio data is stored locally on your machine.  
No account. No cloud sync. No data sent to any server.
